Problem #1: Manipulating Lists

These problems should each be solved by writing a single, recursive function. ANOTHER IMPORTANT RULE! You may not define any helper functions for this part of the assignment -- the answer to each problem should be a single recursive function.
  1. Write a function doubleAll that accepts an integer list l and returns a new list that is identical to l but with each value doubled. For example, doubleAll([1,2,3]) returns [2,4,6].

  2. Write a function countPos that accepts an integer list l and returns the number of elements of the list that are positive. For example, countPos([1, -4, 0, 4, 8, 0]) returns 3.

  3. Write a function intRange that accepts two integers low and high and returns a list of all integers in the range between low and high inclusive. For example, intRange(2, 5) returns [2,3,4,5].

  4. Write a function merge that accepts two integer lists l1 and l2 that are each sorted from least to greatest and produces a new list that contains the elements of both lists, also sorted from least to greatest. Note that duplicates are allowed, both within and across lists. For example, merge([1,2,4], [2,3,3,5]) returns [1,2,2,3,3,4,5].

Problem #2: Turtle Graphics

The program simply commands a turtle to move around a canvas, and the path of the turtle is the shape that is drawn. Python has a library called turtle that supports turtle graphics. You can see its documentation here.

There are lots of commands, but we will only be using two of them: forward(d) moves the turtle forward for distance d (a number), and left(a) turns the turtle left by a degrees.
