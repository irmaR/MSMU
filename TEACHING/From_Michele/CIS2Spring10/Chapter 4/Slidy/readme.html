Browsing the Slides

  • Enable JavaScript for slide navigation
  • Advance to next slide with a mouse click
  • Move forward/backward between slides with space bar, Pg Up and Pg Dn, left-arrow and right-arrow keys
  • Home key for first slide, End key for last slide
  • Function F11 to go full screen and back
  • The A key toggles display of current vs all slides
    • Use the A key when you want to view or print all slides
  • The C key displays the Table of Contents and any key hides it
  • The F key toggles the display of the footer
  • The K key toggles the use of mouse click to advance to the next slide
  • Font sizes automatically adapt to browser window size
    • use < and > keys for additional control (or - and + keys on number pad)
    • The S key makes the font size smaller; the B key makes the font size bigger
  • Switching off JavaScript reveals all slides
  • Not all keys are supported in all browsers, as browsers may reserve some keys for browser control and this varies from one browser to the next

The slides are in HTML Slidy format.
